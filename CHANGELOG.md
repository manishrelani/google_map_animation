# Changelog

All notable changes to this project will be documented in this file.

## [0.0.2] - 2026-01-09

### Added
- `useBearingFromMarker` option for bearing handling.

### Fixed
- Marker jumping during frequent updates.
- Unwanted/redundant calls causing unnecessary updates.


 
## [0.0.1] - 2025-07-07

### Added
- Initial release of Google Map Animation library
- Support for animated polylines on Google Maps
- Support for animated markers on Google Maps
- Smooth transitions and customizable animation effects
